# 链家网新房数据可视化及预测系统 python737

## 项目概述

本项目是一个基于Python语言及Flask框架，运用机器学习算法实现链家网新房数据可视化与预测的系统。系统通过爬虫技术获取数据，使用数据库存储，并通过Web界面展示数据分析结果。

## 技术栈

- **爬虫**: `requests`
- **数据库**: `MySQL`
- **Web框架**: `Flask`
- **机器学习预测算法**: `scikit-learn`
- **可视化**: `Echarts`

## 功能模块

- 数据爬取：全国新房数据爬取
- 数据存储：数据存储到MySQL数据库
- 数据可视化：使用Echarts进行数据可视化展示

### 数据字段

- 房名
- 封面
- 市区
- 地区
- 详细地址
- 房型面积
- 是否具有预售证
- 装修情况
- 开发公司
- 房屋类型
- 交房时间
- 开盘时间
- 标签
- 总价区间
- 售房情况
- 详情链接

## 系统角色

- 数据爬虫：负责链家新房数据的爬取
- 数据库：存储爬取到的数据
- Web服务器：提供数据可视化与预测服务
- 客户端：用户通过Web界面查看数据和分析结果

## 运行环境

- Python 3.x
- Flask
- MySQL
- scikit-learn
- Echarts

## 部署步骤

1. 安装Python及相关依赖
2. 配置MySQL数据库
3. 部署Flask应用
4. 运行爬虫脚本
5. 启动Web服务
